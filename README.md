# SN-Lighting-Mod
Returns old lighting from version 1.0.0 to version 1.7.0.2 of the Secret Neighbor game
---------

# Installation Instructions

!!! First you need to find the anti-cheat bypass and run the mod with it. !!!

1. Install MelonLoader (v0.7.3).
2. Put `light_sn.dll` file into the `Mods` folder.
3. Put `MelonPreferences.cfg` file into the `UserData` folder.
4. Create a folder with the name `LightingBridge` in the `UserData` folder.
5. Put `lighting.json` file into the `LightingBridge` folder.
6. Run the game. The lighting will be applied directly in the match itself.
